## 项目概览
LinguaContext 是一个基于 Chrome Manifest V3 的沉浸式英语学习扩展，利用 AI 提供语境感知的单词解释。

### 核心功能模块
1.  **语境解释 (Context-Aware Explanation)**
    *   **触发**: 用户划词后通过 Content Script 发送 `FETCH_EXPLANATION` 消息。
    *   **处理**: Background Service Worker 调用 `callLLMStreaming` (火山引擎 LLM)。
    *   **反馈**: 通过流式消息 (`STREAM_UPDATE`) 实时将解释推送到前端。
2.  **语音朗读 (TTS)**
    *   **机制**: 使用 `offscreen` document (`src/offscreen/offscreen.html`) 来播放音频，绕过 Service Worker 无法播放音频的限制。
    *   **服务**: 集成火山引擎 TTS API。
3.  **生词本与同步 (Vocabulary & Sync)**
    *   **存储**: 本地使用 `chrome.storage.local`。
    *   **同步**: 集成 Google Drive API (`src/background/google-drive.js`)，支持多端数据合并与冲突处理。
    *   **界面**: 独立的 Dashboard 页面 (`src/dashboard/dashboard.html`) 用于管理生词。

### 关键代码结构
*   **入口**: `manifest.json` 定义了权限、Background Worker 和 Content Scripts。
*   **后台**: `src/background/service-worker.js` 是消息处理中心，协调各个模块。
*   **服务**: `src/services/` 目录下封装了 LLM 和 TTS 的 API 调用逻辑。
